SliperShop 0.1 alpha - это скрипт интернет - магазина написанный  на PHP 
использующий текстовые файлы для хранения информации, прост в установке,
настройке и использовании. Интуитивный, дружественный интерфейс
пользовательской части магазина.

Основные функции скрипта: 
*Неограниченное количество уровней категорий товаров.
*Возможность создания перекресных уровней.
*Возможность добавления полного описания к товару. 
*Возможность добавления изображения к товару.
*Вся база хранится в обычных текстовых файлах.
*Изображения и описание к разделам.
*Вывод по несколько товаров на станицу.
*Выбор количества столбцов товара при выводе.
*Неограниченное число полей при оформлении заказа.

Административная часть отсутствует, будет реализована в следующей версии. 


Как это работает:
Главной частью магазина является скрипт SHOP.PHP
База товаров находится в папке "data\shop\"
Файл shop.txt - список основных категорий товара.
Остальные файлы это базы подкатегорий...

Структура shop.txt:
Первая строка имеет следующий вид 

ххх|Раздел1::Фаил1::Картинка раздела 1|......|РазделN::ФаилN::Картинка раздела N|

здесь ххх-описание раздела
      Раздел1-название раздела
      Фаил1-файл базы товаров раздела
      Картинка раздела 1-название картинки для раздела
                         (находятся в папке "data\upimages\")

Первые строки файлов подразделов имеют такойже формат, только там уже не "Раздела",
а "Подразделы". Следующие строки файлов подразделов содержат описание товара и имеют
следующий вид (одна строка соответствует данным об одном товаре):

ID|Цена|Название товара|Краткое описание|Полное описание|Картинка товара|

При отсутствии полного описания товара отображается краткое...

Файлы:
category_payment.txt-способы оплаты и доставки (выводятся при заказе)
config_shop.php-файл конфигурации (раскоментирован)
polia.php-поля при заказе (содержимое интуитивно понятно)
zak.txt-файл содержащий заказы
